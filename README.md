# .github

Organization-level defaults and community health files for **[The AI Theorem](https://github.com/the-ai-theorem)**.

## Contents

- **[`profile/README.md`](./profile/README.md)** — the public organization profile rendered at <https://github.com/the-ai-theorem>. **English is the source of truth**; edit it first, then update translations.
- **[`profile/i18n/`](./profile/i18n/)** — translations of the profile (12 languages). Each file carries a language switcher linking to all others.

## Languages

English (source) · 한국어 · 中文 · 日本語 · Deutsch · Français · Español · Português · Русский · Italiano · हिन्दी · العربية

## Updating

1. Edit **`profile/README.md`** (English).
2. Propagate changes to the matching files under **`profile/i18n/`**, keeping the language switcher, badges, tables, and code blocks identical in structure.
3. Keep brand and technical terms in English: `The AI Theorem`, repo names, `Canon`, `.theorem/`, `Single Source of Truth`, product names, and the eight principle names.
